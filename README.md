A recon(struction) of PvZ, which aims to add more features...than usual.

### Base Features
- Overhauled rendering system, allowing the choosing of different backends.
- Replaced/Updated API to support newer hardware.
  * DSound has been replaced by OpenAL-soft
  * BASS has been updated to 2.4
  * TTF font rendering is now done by the TrueType library
  * WinAPI code was replaced with SDL3 alternatives where possible (SEHCatcher was built with WinAPI.)
## Modding
The project has the goal of making developing mods easier and making them more accesible
